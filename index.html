Welcome to Inkstand

Inkstand is a lightweight microservice container for easing the creation of standalone REST services. It provides full CDI 1.1 and Jax-RS support. It is built on top of the Weld CDI Container, Undertow and RestEasy.

Features

Inkstand provides some extra features

  • JCR Support for providing content served by a JCR Repository
  • LDAP based authentication
  • The testing framework Scribble

Origin

Inkstand is a complete reimplementation of the initial idea was introduced by John Ament: Announcing Hammock in February 2014.
